# rubydown installation
```
git clone git://github.com/sciruby-jp/rubydown
cd rubydown
rake build
rake install
```

# Converting (and executing) rbmd to html
```
/usr/local/bundle/bin/rubydown -i INPUTRBMD -e YOURERB -o OUTPUT
```
